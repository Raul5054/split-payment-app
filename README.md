# Debt Manager

A simple Flutter app to manage shared expenses between friends, roommates, and groups.

## Features

-  Add expenses with description and amount
-  Split bills equally between people
-  Track who paid and who owes
-  Simple debt calculations
-  User-friendly interface

## Getting Started

This project is built with:
- Flutter (Dart)
- Firebase (coming soon)
